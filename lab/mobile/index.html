What is Brand Experience Design? It's a combination of UX (User Experience), UI Design (User Interface), and Brand Design. A Brand Experience Designer is like a tailor, skilled at stitching together many different elements. The goal is to provide good design that fits well and feels right. Brand Experience Designers present users/customers with a unified set of relevant elements, so experiences are logical, make sense and are easy to navigate.

About this Beta

I'm thrilled to be working with this code to build mobile websites. The jQuery project is great for the internet because the project strives to develop widely used standards, while also maintaining high-quality, efficient design. This sample mobile site is something that I'm tinkering with in order to learn more about how to possibly set these types of sites up.

I suppose on of the biggest challenges will be trying to figure out how I can use the code to design better websites as well as web apps... and not let the conventions that are so easy to manipulate here close me off to pushing toward new creative ideas and interactions.
